# IntelliSpend 💸

IntelliSpend is a modern personal finance analytics web application that helps users track income, manage expenses, and understand their financial health through real-time calculations and interactive dashboards.

🚀 Live Application:  
https://intellispent-6au5dlirx-priyankabc1910s-projects.vercel.app/

## Features

- Add monthly income
- Add multiple expense categories
- Real-time savings calculation
- Savings ratio analysis
- Interactive spending analytics dashboard
- Clean and responsive UI

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Vercel (Deployment)

Built and deployed independently.
